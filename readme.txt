== Description ==
A small widget for display the latest MeiPian posts.

== Installation ==
= Install From WordPress Repository =
1. Go to Plugins, find and Install & activate "Sysbox - Latest MeiPian Post List Widget"
2. To use go to widgets and drag & drop the "Sysbox - Latest MeiPian Post List Widget" to the selected section
3. The widget will ask you to input a title, no of posts and the url of the meipian account.
4. Save the content for the selected page will now be included in the widget section

= Install Via Upload =
1. Download the File from https://wordpress.org/plugins/meipian-latest-posts-list-widget/
2. Upload, Install & activate "Sysbox - Latest MeiPian Post List Widget"
3. To use go to widgets and drag & drop the "Sysbox - Latest MeiPian Post List Widget" to the selected section
4. The widget will ask you to input a title, no of posts and the url of the meipian account.
5. Save the content for the selected page will now be included in the widget section
